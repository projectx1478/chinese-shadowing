# 听写跟读（ディクテーション＆シャドーイング練習）

## 概要

中国語のディクテーション（書取）とシャドーイングを組み合わせた、個人学習用の中国語学習アプリです。AIが生成する会話文をもとに、聞き取り・書き取り・音読・発音採点までを一連の流れで練習できます。

## 主な機能

- AIによる会話文自動生成（レベル・トピック・HSK重要単語優先などを指定可能）
- ディクテーション（書取）：音声を聞いて中国語を書き取り、LCSアルゴリズムで一致率を採点
- シャドーイング：手本音声に続いて音読・追読・録音し、AIが発音・声調を評価
- 声調（ピッチ）グラフ：録音の音の高さを解析し、期待される声調概形と比較表示
- AIコーチモード：復習キュー（簡易SRS）に基づく今日の学習メニューを自動生成、コーチからの声掛け・締めコメント付き
- 学習履歴・分析：スコア推移グラフ、AIによる発音傾向分析（声調別ミス集計）、トピック別レーダーチャート
- 苦手リスト・カスタム文セットの保存
- Google認証によるマルチデバイス同期（Firestore）

## 使用技術

- フロントエンド：Vanilla JavaScript / HTML / CSS（単一 `index.html`、ビルドツール不使用）
- バックエンド：Firebase Authentication（Google認証）、Firestore（データ永続化）
- AIプロキシ：Cloudflare Workers（Gemini APIキーをクライアントから隠蔽、Firebase IDトークンをJWKS方式で検証）
- AI：Google Gemini API（会話文生成・発音採点解説・AIコーチのメッセージ生成）
- 音声：Web Speech API（TTS）、MediaRecorder API（録音）、Web Audio API（声調ピッチ解析）
- その他：Canvas API（グラフ描画）

## 動作環境

- スマートフォン（メイン利用環境）
- タブレット
- 対応ブラウザ：モバイルSafari / Chrome（HTTPS環境が必須。マイク・音声合成機能のため）
